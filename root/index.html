Project Overview

Cyber Incident Response OpenEnv is a long-horizon simulation environment designed for evaluating AI agents in real-world cybersecurity defense scenarios.

The environment models a Security Operations Center (SOC) where an AI defender must detect, investigate, and contain cyber attacks launched by an automated adversary. The system supports multi-step decision making, dynamic attack propagation, and measurable defensive outcomes.

This project follows the OpenEnv standard for interactive AI environments and demonstrates how large language models can assist in real-time cyber defense.

Why This Environment Matters
  • ⚔️ Simulates real red-team vs blue-team cyber conflict
  • 🧠 Designed for evaluating autonomous AI security agents
  • 🔄 Long-horizon decision making instead of single-step tasks
  • 📊 Reward shaping and task grading for agent benchmarking
  • 🌐 Deployable through Hugging Face Spaces using Docker
  • 🔌 Compatible with OpenEnv standard interfaces
What Makes This Unique
  • ⚡ Real-time adversarial simulation (not static datasets)
  • 🧩 Multi-step decision environment (not single-shot tasks)
  • 🤖 Designed specifically for LLM-based agents
  • 📈 Continuous reward signals for learning evaluation
  • 🔁 Multiple attack scenarios for trajectory diversity

How to Try the Simulator
  1. Select a difficulty level (Easy, Medium, Hard).
  2. Click Start Game to initialize the environment.
  3. Investigate hosts to detect suspicious activity.
  4. Isolate infected systems to prevent lateral movement.
  5. Patch vulnerable machines to restore the network.

Your goal is to protect critical servers before the attacker compromises the network.
